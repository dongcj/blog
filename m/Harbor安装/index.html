1.2. 高级配置（可选）

1.2.1. 配置 Clair 和 Notray

clair 是 coreos 开源的容器漏洞扫描工具，在容器逐渐普及的今天，容器镜像安全问题日益严重。clair 是目前少数的开源安全扫描工具，主要提供 OS（centos，debian，ubuntu 等）的软件包脆弱性扫描。

Notary 是一套 docker 镜像的签名工具， 用来保证镜像在 pull，push 和传输工程中的一致性和完整性。 避免中间人攻击，避免非法的镜像更新和运行。

2. watchdog

3. harbor 同步设置

3.1. 方案

CS (registry.svicloud.com) — 全同步 —> SZ (ci.svicloud.com) — 部分同步 —> FR (registry.svicloud.com)

3.2. 关于国内外域名相同的方案

方法有二种： 1. 在深圳 harbor 服务器上，设置 DNS 为 1.1.1.1。 这样的话，深圳解析到的地址和国外解析到的地址一样，这样同步时就可以从深圳同步向法国了（推荐方法） 2. 在 ui 这个容器中设置 registry.svicloud.com 的 IP 地址（未测试是否完全可用）
